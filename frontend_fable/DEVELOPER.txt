================================================================================
FWIP.SK — frontend_fable — DOKUMENTÁCIA PRE DEVELOPERA
================================================================================

Posledná aktualizácia: jún 2026
Stack: Vite 5 + React 18 + TypeScript + Tailwind CSS v3 + Framer Motion
       + React Router v6. Dáta zo Strapi CMS (REST).

--------------------------------------------------------------------------------
1. SPUSTENIE
--------------------------------------------------------------------------------
cd frontend_fable
npm install
npm run dev        -> http://localhost:5174  (port je fixný, strictPort)
npm run build      -> tsc typecheck + produkčný build do dist/
npm run preview    -> náhľad produkčného buildu na 5174

.env premenné (pozri .env.example):
  VITE_STRAPI_URL          adresa Strapi (produkcia: Railway inštancia)
  VITE_GA_MEASUREMENT_ID   GA4 ID (G-XXXX...). Prázdne = žiadne trackovanie.

3. TOK DÁT (ako to celé funguje)
--------------------------------------------------------------------------------
Stránka -> hook (napr. useProducts) -> useStrapi(path, transform, fallback)
  -> lib/strapi.ts strapiGet() -> VITE_STRAPI_URL + path
  -> fixUrls() prepíše každé "url" pole: privátny R2 bucket -> verejný
     https://pub-....r2.dev (bez toho médiá nefungujú!)
  -> transform() znormalizuje Strapi payload na app model
  -> pri chybe ALEBO prázdnej odpovedi sa vráti slovenský fallback
     zo src/data/ a isFallback=true. Stránka teda NIKDY nie je prázdna.

Obsah ktorý NIE JE v Strapi (hero texty, kroky fallback, B2B body, navigácia,
footer skupiny, 404...) -> src/data/static.sk.ts.

CAPS texty z CMS sa zobrazujú cez sentenceCase() — obsah v CMS sa NEMENÍ.
Farby z CMS (príchute, promo, places) idú cez inline style — to je jediná
povolená výnimka z pravidla "žiadne raw hex v komponentoch".

4. SEO + GOOGLE
--------------------------------------------------------------------------------
- usePageMeta (src/hooks/usePageMeta.ts): každá stránka si nastaví <title>,
  meta description, OG tagy a canonical. Texty sú v PAGE_META.
- index.html: JSON-LD Organization (telefón, e-mail, sociálne siete).
- public/sitemap.xml: statická. PRI PRIDANÍ PRÍCHUTE V STRAPI treba ručne
  doplniť /zmrzlina/<slug> (slug = názov bez diakritiky, medzery -> pomlčky).
- public/robots.txt: Allow všetko + Sitemap odkaz.
- POZOR: je to SPA (client-side rendering). Googlebot JS renderuje, ale pre
  najlepšie indexovanie zvážte pri nasadení prerendering (napr.
  vite-plugin-ssr / prerender.io) alebo aspoň serverový fallback na index.html
  pre všetky routy (nutné aj pre React Router — 404 na serveri = zle).
- Google Search Console: overiť doménu, odoslať sitemap.xml.

--------------------------------------------------------------------------------
5. TRACKOVANIE (GA4)
--------------------------------------------------------------------------------
src/lib/analytics.ts:
- gtag.js sa NAČÍTA AŽ keď používateľ klikne OK na cookie lište
  (localStorage 'simpleCookieConsent' = 'accepted') — GDPR friendly.
- Pri ďalších návštevách s uloženým súhlasom sa inicializuje hneď (App.tsx).
- Page-view sa posiela pri každej zmene routy (SPA tracking).
- anonymize_ip zapnuté.
- Aktivácia: nastav VITE_GA_MEASUREMENT_ID v .env a rebuildni.
  Prázdne ID = celý modul je no-op, nič sa nenačíta.

--------------------------------------------------------------------------------
6. ANIMÁCIE — PRAVIDLÁ
--------------------------------------------------------------------------------
- Všetky varianty v src/design-system/animations.ts.
- Scroll-triggered: VŽDY whileInView + viewport={{ once:true, margin:'-80px' }}
  priamo na motion elemente. NIKDY useInView s ref na element, ktorý sa
  mountuje až po načítaní dát — observer sa nepripojí a sekcia ostane
  neviditeľná (historická chyba, už opravená).
- Každý motion komponent berie varianty cez useMotionSafe() — automaticky
  vypne pohyb pri prefers-reduced-motion. Videá sa vtedy neautoplayujú.

7. ČASTÉ ÚPRAVY — KDE SIAHNUŤ
--------------------------------------------------------------------------------
Zmena farby/fontu          -> tailwind.config.ts (+ tokens.ts)
Nový text mimo CMS         -> src/data/static.sk.ts
Nová príchuť               -> pridať v Strapi (ice-creams) + sitemap.xml;
                              fallback voliteľne do flavors.sk.ts
Nová stránka               -> src/pages/X.tsx + route v App.tsx + odkaz
                              v static.sk.ts NAV/FOOTER + PAGE_META + sitemap
Nová Strapi kolekcia       -> typ do types/index.ts + hook do src/hooks/
                              (vzor: useZariadenia) + fallback
Kontaktný formulár polia   -> ContactForm.tsx + types (POZOR na `emial`)
Cookie texty               -> CookieConsent.tsx
Navigačné položky          -> static.sk.ts -> NAV

--------------------------------------------------------------------------------
8. ZNÁME ZVLÁŠTNOSTI (nie sú to chyby)
--------------------------------------------------------------------------------
- Strapi pole `emial` (preklep v schéme) — musí ostať, inak POST zlyhá.
- Kolekcia krokov sa volá `stepss` (dvojité s) — tak je to v Strapi.
- Logo zo Strapi je teal PNG — v navbare hore a vo footri sa invertuje
  na bielo cez CSS filter (brightness-0 invert).
- Médiá chodia z privátneho R2 bucketu -> prepis na pub-...r2.dev
  v lib/strapi.ts. Keď sa zmení bucket, zmeň konštanty R2_PRIVATE/R2_PUBLIC.
- Porovnávacia tabuľka: 4. typ v CMS nemá meno (name=null) — to JE fwip
  stĺpec, renderuje sa logom.
